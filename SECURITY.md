# Security and privacy

## Credentials

Do not commit API keys. DOVE supports environment-variable references in model
configuration and session-only password input in DOVEboard.

## Sensitive benchmark content

Result JSON can contain question text, raw model responses, and metadata. Private
corpora and result directories are ignored by default, but repository operators
remain responsible for access control, redaction, retention, and secure deletion.

## Reporting a vulnerability

Before publishing this repository, replace this section with a private security
contact or enable GitHub private vulnerability reporting.
